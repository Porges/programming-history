## Unicode support

<dl>
<dt>Identifiers</dt>
<dd>
    <ul>
    <li>✓: follows [UAX-31](http://www.unicode.org/reports/tr31/).</li>
    <li>(Blank): allows Unicode in some form.</li>
    <li>✗: does not allow Unicode.</li>
    </ul>
</dd>
</dl>
